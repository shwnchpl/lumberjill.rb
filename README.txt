Lumber Jill
-----------

A simple Telegram logging bot, intended for use with maul.rb.

Instructions
------------

- Create a Telegram bot using @BotFather to get an API token.
    * Ensure group "Privacy Mode" is not enabled for the bot.
- Note the ID of the Telegram account to be used as admin.
